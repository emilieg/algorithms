#Stacks

##About
Uses the **L**ast **I**n **F**irst **O**ut (LIFO) approach => items are added to the end and are removed from the end

##When to use
- For Example: the 'undo' functionality undoes the last step that was made

##Methods in JavaScript
- **push()** & **pop()**
