#Queues

##About
Uses the **F**irst **I**n **F**irst **O**ut (FIFO) approach => items are added to the end and are removed from the beginning

##When to use
- For Example: processing work orders in order received, standing in line

##Methods in JavaScript
- **push()** & **shift()**
